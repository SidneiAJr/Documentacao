# Curso Oracle Next Education One

## Descrição

Curso **totalmente online**, com **suporte via Discord da Alura**, feito em parceria com a **Oracle**. A formação oferece acesso gratuito a conteúdo educacional focado principalmente em **Java** e **APIs**, com uma certificação disponível no ambiente da Oracle.

- **Bolsa da Oracle**: O curso foi disponibilizado gratuitamente para os participantes por meio de uma bolsa oferecida pela Oracle, possibilitando o acesso ao conteúdo sem custo.

## Pontos Positivos

- **Certificação Gratuita**: Ao final do curso, você tem a opção de obter **certificação gratuita** diretamente no ambiente da Oracle, o que é uma vantagem para quem deseja validar o aprendizado.
  
- **Suporte e Comunidade**: O curso conta com **suporte via Discord** da Alura, o que facilita a interação com outros alunos e instrutores.

- **Bom para Iniciantes**: Se você está começando a aprender **Java** ou **APIs**, o curso pode ser um bom ponto de partida, fornecendo os fundamentos necessários para avançar no aprendizado.

## Pontos Negativos

- **Conteúdo Repetitivo**: Para quem já possui conhecimento prévio em Java ou desenvolvimento de APIs, o curso pode parecer repetitivo e básico, abordando conceitos simples de maneira excessiva.

- **Falta de Profundidade**: O curso é adequado para iniciantes, mas para quem já tem alguma experiência, ele pode ser considerado **superficial**, não aprofundando em tópicos mais avançados ou melhores práticas de mercado.

## Conteúdo do Curso

O curso cobre diversos tópicos básicos relacionados a **Java** e **APIs**, incluindo:

### 1. **Introdução ao Java**
   - Estrutura básica de um programa Java
   - Tipos de dados e variáveis
   - Controle de fluxo (if, switch, loops)
   - Funções e métodos em Java

### 2. **Java com APIs**
   - Como trabalhar com **APIs RESTful** em Java
   - Consumo de APIs externas
   - Noções de **JSON** e como manipulá-lo em Java
   - Ferramentas para testar APIs (como o **Insomnia**)

### 3. **Trabalhando com API com Insomnia**
   - Uso da ferramenta **Insomnia** para testar e desenvolver APIs
   - Envio de requisições e análise de respostas (GET, POST, PUT, DELETE)
   - Criação de scripts e automações simples para consumir APIs

### 4. **Ferramentas Complementares**
   - **Insomnia**: Ferramenta gráfica para testar APIs REST e analisar dados de resposta de forma prática.

---

## Conclusão

Embora o **Curso Oracle Next Education One** seja uma boa oportunidade para quem está iniciando na área de **Java** e **APIs**, ele pode não ser o mais indicado para quem já tem conhecimento na área, pois o conteúdo é bastante básico e repetitivo. Para quem deseja obter a **certificação gratuita** da Oracle e busca uma introdução ao mundo de APIs e Java, o curso é válido.
