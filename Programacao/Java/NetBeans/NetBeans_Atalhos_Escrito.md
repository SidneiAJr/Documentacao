# Atalhos de Código no NetBeans

- **sou** → `System.out.println();`
  - **Explicação**: Gera um comando de impressão no console para depuração ou exibição de mensagens.

- **soutv** → `System.out.println("variável: " + variavel);`
  - **Explicação**: Gera um `System.out.println` com a variável dentro de uma mensagem de log, facilitando a exibição de valores de variáveis.

- **psvm** → `public static void main(String[] args) { }`
  - **Explicação**: Gera automaticamente o método `main`, ponto de entrada do programa em Java.

- **fori** → `for (int i = 0; i < ; i++) { }`
  - **Explicação**: Gera um laço de repetição `for` com uma variável de controle `i`.

- **foreach** → `for (Tipo variavel : colecao) { }`
  - **Explicação**: Gera um laço `foreach` para iterar sobre uma coleção (como uma lista, array, etc.), sem a necessidade de controlar o índice.

- **try** → `try { } catch (Exception e) { } finally { }`
  - **Explicação**: Gera uma estrutura de tratamento de exceções com bloco `try`, `catch` para capturar exceções e `finally` para execução final, independentemente de erros.

- **tryf** → `try { } catch (Exception e) { }`
  - **Explicação**: Gera um bloco de código `try-catch`, sem o bloco `finally`, utilizado para capturar e tratar exceções.

- **if** → `if () { }`
  - **Explicação**: Gera uma estrutura condicional `if`, que executa um bloco de código se a condição for verdadeira.

- **else** → `else { }`
  - **Explicação**: Gera a parte do código que será executada se a condição do `if` for falsa.

- **elseif** → `else if () { }`
  - **Explicação**: Gera uma condição adicional após um `if` para testar uma outra condição, caso a anterior seja falsa.

- **while** → `while () { }`
  - **Explicação**: Gera um laço de repetição `while` que executa enquanto a condição for verdadeira.

- **do** → `do { } while ();`
  - **Explicação**: Gera um laço `do-while`, que garante que o bloco de código será executado pelo menos uma vez antes de verificar a condição.

- **switch** → `switch () { case : break; }`
  - **Explicação**: Gera a estrutura `switch`, que permite realizar diferentes ações dependendo do valor de uma variável.

- **sysout** → `System.out.println();`
  - **Explicação**: Gera o comando `System.out.println`, usado para imprimir algo no console (sinônimo de `sou`).

- **sysin** → `Scanner sc = new Scanner(System.in);`
  - **Explicação**: Gera uma instância de `Scanner` para ler dados da entrada padrão (teclado).

- **javadoc** → `/** documentation */`
  - **Explicação**: Gera um template de comentário Javadoc, utilizado para documentar classes, métodos e variáveis em Java.

- **main** → `public static void main(String[] args) { }`
  - **Explicação**: Gera o método `main`, que é o ponto de entrada do programa em Java.

- **ctor** → `public NomeClasse() { }`
  - **Explicação**: Gera o construtor padrão de uma classe, que é usado para inicializar objetos dessa classe.

- **class** → `class NomeClasse { }`
  - **Explicação**: Gera uma declaração de uma nova classe com o nome especificado.

- **interface** → `interface NomeInterface { }`
  - **Explicação**: Gera uma declaração de uma interface, que é um tipo de classe abstrata usada para definir contratos que as classes podem implementar.

- **package** → `package nome.do.pacote;`
  - **Explicação**: Gera a declaração de um pacote, que é uma maneira de organizar classes e interfaces dentro de um projeto Java.

- **import** → `import nome.do.pacote;`
  - **Explicação**: Gera a declaração `import`, que permite importar outras classes ou pacotes para o código atual.

- **throw** → `throw new Exception("mensagem");`
  - **Explicação**: Gera a instrução `throw` para lançar uma exceção explicitamente no código.

- **throws** → `public void metodo() throws Exception { }`
  - **Explicação**: Gera a declaração `throws`, que especifica que um método pode lançar exceções que precisam ser tratadas por quem chama o método.
