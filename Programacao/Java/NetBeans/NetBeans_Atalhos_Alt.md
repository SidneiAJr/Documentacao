# Atalho Alt + Insert no NetBeans

O atalho **Alt + Insert** no NetBeans abre o menu de **geração de código** dentro de uma classe. Ele é muito útil para criar rapidamente código repetitivo sem precisar digitar tudo manualmente.

## Funcionalidades principais:

- **Construtor (Constructor)**
  - Gera automaticamente o construtor da classe, com todos ou alguns atributos.
  - Permite inicializar os campos da classe de forma rápida.

- **Getters e Setters**
  - Gera métodos **get** e **set** para os atributos da classe.
  - Útil para encapsulamento e acesso seguro aos campos privados ou protegidos.

- **Override Methods**
  - Gera métodos que você pode sobrescrever (override) de superclasses ou interfaces implementadas.
  - Facilita implementar métodos de interfaces ou classes abstratas.

- **Implement Methods**
  - Mostra uma lista de métodos obrigatórios de interfaces ou classes abstratas que ainda não foram implementados.
  - Permite implementar rapidamente os métodos que o Java exige.

- **toString()**
  - Gera automaticamente o método `toString()`, retornando uma representação textual da classe, geralmente incluindo os valores dos atributos.

- **hashCode() e equals()**
  - Gera os métodos `hashCode()` e `equals()` para comparação de objetos e uso em coleções.

- **Delegating Methods**
  - Permite criar métodos que delegam chamadas a um objeto interno.
  - Útil para composição e design orientado a objetos.

- **Insert Code**
  - Opção geral para inserir rapidamente blocos de código comuns, como construtores, getters/setters e overrides.

## Como usar:
1. Coloque o cursor dentro da classe onde quer gerar o código.
2. Pressione **Alt + Insert**.
3. Escolha o que deseja gerar no menu que aparecerá.
