# Funcionalidades adicionais do Alt + Insert no NetBeans

Além de construtores, getters, setters, overrides e implementações de métodos, você também pode:

- **Override/Implement abstract methods**
  - Mostra todos os métodos abstratos ou de interfaces que ainda não foram implementados na classe.

- **Generate Comments**
  - Cria comentários automáticos ou Javadoc para classes, métodos e atributos.

- **Generate Serializable UID**
  - Quando a classe implementa `Serializable`, permite gerar um `serialVersionUID` automaticamente.

- **Add Field from Constructor**
  - Permite criar atributos automaticamente a partir do construtor ou de parâmetros de método.

- **Delegate Methods**
  - Gera métodos que delegam chamadas a outro objeto que a classe possui internamente.

- **Insert code templates**
  - Permite acessar snippets ou templates de código configurados no NetBeans (como `sout`, `fori`, etc.).

- **Custom Code Generation**
  - Alguns plugins ou configurações adicionais podem adicionar opções extras ao menu Alt + Insert.

- **Event Handlers**
  - Para classes que lidam com GUI (Swing/JavaFX), pode gerar rapidamente métodos de tratamento de eventos, como `actionPerformed`.

- **Main Method**
  - Permite gerar rapidamente o método `public static void main(String[] args)` se ainda não existir.
