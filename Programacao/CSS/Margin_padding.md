# Diferença entre padding e margin no CSS

- ***padding → Espaço interno entre o conteúdo do elemento e sua borda.***

- ***margin → Espaço externo entre o elemento e os outros elementos ao redor.***

+-----------------------+
|       padding         |
|   +---------------+   |
|   |   conteúdo    |   |
|   +---------------+   |
+-----------------------+
        margin
