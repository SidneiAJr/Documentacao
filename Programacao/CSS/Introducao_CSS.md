# Tutorial de CSS:

## Resetando Globalmente para todos os navegadores
```css
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
    outline: none;
    text-decoration: none;
}
```

## Colocando Linear gradiente no corpo do html:

```css
body {
    background: linear-gradient(to left, #6a11cb, #2575fc);
}
```
