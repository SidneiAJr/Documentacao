# Exemplo de Codigo em C:

## Soma de Int com Int
```C
#include <stdio.h>

int main() {
    int a = 5;
    int b = 5;
    int soma = a+b;
    printf("Soma %d \n",soma);
}
```

## Comparação de Int com Double
```C

#include <stdio.h>

int main() {
    int a = 5;
    double b = 5.5;
    int comparador = a == b;
    printf("Comparação %d \n",comparador);
    
}
```
