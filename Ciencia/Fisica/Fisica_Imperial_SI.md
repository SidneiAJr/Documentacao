# Sistema Imperial vs Sistema Internacional (SI)

## Conversão do Sistema Internacional para o Sistema Imperial

### 1. **BTU para Watt (W)**

- **BTU (British Thermal Unit)** é uma unidade de energia usada no Sistema Imperial, especialmente em contextos de aquecimento e refrigeração.
- **Watt (W)** é uma unidade de potência do Sistema Internacional, usada para medir a taxa de consumo ou geração de energia.

**Fórmula de conversão**:

\[
1 \, \text{BTU/h} \approx 0.293071 \, \text{W}
\]

### 2. **Quilograma (kg) para Libra (lb)**

- **Quilograma (kg)** é a unidade de massa do Sistema Internacional (SI).
- **Libra (lb)** é a unidade de massa do Sistema Imperial.

**Fórmula de conversão**:

\[
1 \, \text{kg} \approx 2.20462 \, \text{lb}
\]

### 3. **Kilômetro (km) para Jarda (yd)**

- **Kilômetro (km)** é a unidade de comprimento do Sistema Internacional.
- **Jarda (yd)** é a unidade de comprimento do Sistema Imperial.

**Fórmula de conversão**:

\[
1 \, \text{km} = 1093.6133 \, \text{yd}
\]

---

## Resumo das Conversões

| **Unidade (SI)**     | **Unidade (Imperial)** | **Fator de Conversão**      |
|----------------------|------------------------|-----------------------------|
| **1 BTU/h**          | **0.293071 W**          | Energia/ Potência térmica    |
| **1 kg**             | **2.20462 lb**          | Massa                       |
| **1 km**             | **1093.6133 yd**        | Comprimento                 |

---

### Considerações Importantes:

- O **Sistema Internacional** (SI) é amplamente utilizado em contextos científicos, acadêmicos e industriais ao redor do mundo, enquanto o **Sistema Imperial** é utilizado principalmente em alguns países, como os **Estados Unidos**.
- As conversões entre os sistemas são essenciais para garantir a **interoperabilidade** de dados, dispositivos e sistemas em contextos internacionais.
